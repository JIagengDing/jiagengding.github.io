7z

添加文件

7z a test.7z test_folder/

移除文件

7z d test.7z test_file

7z d -r test.7z *.png

解压

7z x test.7z [-o test_folder]

zip

压缩

zip -r test.zip test_folder

移除文件

zip -d tset.zip test.png

解压

unzip test.zip
