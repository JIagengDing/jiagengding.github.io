简单使用

初始化或克隆

使用命令git init 来初始化一个文件夹，或者使用命令git clone host:/path/to/repo从远程服务器克隆一个仓库。

添加和提交更改

  1. git add . 将工作目录中的全部更改提交到暂存区INDEX。
  2. git commit -m "更改信息" 用来将INDEX中的更改提交到HEAD。

推送到远程仓库

  • 远端没有本地仓库，推送本地仓库使用命令 git remote add origin <server>。
  • 远端已有本地仓库使用命令git push origin <branch name>推送到该分支。

更新仓库到本地

  • git pull 用来获取并合并远端仓库至本地仓库。
  • git merge <branch name> 将远程特定分支合并到本地。

回退本地更改

  • 使用HEAD中内容替换目录文件 git checkout -- <filename>
  • 切换节点 git checkout <commit>
